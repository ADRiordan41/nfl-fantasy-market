# MatchupMarket Brand Assets

Use the SVG files as the source of truth for merchandise, signage, and print work. They are vector files and can scale without quality loss.

- `matchup-market-mark.svg`: full-color sigma/delta mark on a transparent background.
- `matchup-market-mark-flat.svg`: two-color flat mark for simpler printing.
- `matchup-market-mark-white.svg`: single-color white mark for dark garments or dark backgrounds.
- `matchup-market-icon.svg`: square dark-background app/social icon.
- `*-4096.png`: high-resolution raster exports for tools that do not support SVG.
- `*-2048.png`: practical large uploads for most social, marketplace, and print-on-demand tools.
- `*-1024.png`: standard web/social upload size.
- `*-512.png`: compact previews, profile images, thumbnails, and tools with strict limits.

For embroidery, screen printing, vinyl, and large-format printing, send the SVG file when possible.
